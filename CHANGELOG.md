# Change Log

## [Unreleased]

- Updated documentation to reflect current keymap and behavior.
- Changed NAV->INSERT toggle key from `§` to `Space`.
- Added `k` command for "line end + enter INSERT".
- Changed replace-word from `rr` to direct `R`.
- Updated status bar mode colors:
  - `NAV` uses `statusBarItem.warningBackground`
  - `INSERT` uses `statusBarItem.remoteBackground`
