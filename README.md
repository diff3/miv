# MIV - Modern Improved Vi

MIV is a lightweight modal editing extension for VS Code inspired by Vim but designed for IDE workflows.

## Enable MIV

Press:

`ESC`

to enter NAV mode.

## Basic Example

Navigation:

- `a` left
- `d` right
- `w` up
- `s` down

Insert mode:

- `i` insert
- `ESC` return to NAV mode

Search:

- `/text` search forward
- `\\text` search backward
